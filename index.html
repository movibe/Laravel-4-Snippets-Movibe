Laravel 4 Snippets version Movibe

Emmet style Laravel 4 snippets for Sublime Text 2.

I used Laravel 4 source to create nice collection of snippets.

For longer names there are handy tab-trigger aberrations.

So for {{ Form::text($name, $value, $attributes) }} you can just use Form-text.

Installation

Open Sublime Text 2 -> Package Manager: Add Repository => and copy and paste this repository https://github.com/movibe/Laravel-4-Snippets.git

Available at Package Manager! just search for Laravel 4.
